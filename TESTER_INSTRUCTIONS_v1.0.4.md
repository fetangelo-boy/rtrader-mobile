## ШАГ 5: Полный тест функциональности

### 1️⃣ Список чатов
- [ ] Приложение показывает 12 чатов
- [ ] Чаты: Газ/нефть, Продуктовый, Металлы, Сельхоз, Валюта, Новости рынка, Аналитика, Образование, Test Chat, General, Trading Tips, Market Analysis

### 2️⃣ Вход в чат
- [ ] Нажмите на любой чат (например, "Газ/нефть")
- [ ] Откроется экран с сообщениями
- [ ] Видны сообщения от других пользователей

### 3️⃣ Отправка сообщения
- [ ] Внизу экрана есть поле для ввода сообщения
- [ ] Напишите тестовое сообщение
- [ ] Нажмите кнопку отправки (стрелка)
- [ ] Сообщение появляется в чате

### 4️⃣ Ответ на сообщение
- [ ] Нажмите и держите сообщение другого пользователя
- [ ] Появится меню с опциями
- [ ] Выберите "Ответить"
- [ ] Напишите ответ
- [ ] Отправьте

### 5️⃣ Отключение уведомлений (Mute)
- [ ] Нажмите на название чата вверху
- [ ] Найдите опцию "Отключить уведомления" или иконку звука
- [ ] Отключите уведомления для этого чата
- [ ] Убедитесь, что иконка изменилась

### 6️⃣ Включение уведомлений (Unmute)
- [ ] Нажмите на название чата вверху
- [ ] Найдите опцию "Включить уведомления"
- [ ] Включите уведомления
- [ ] Убедитесь, что иконка вернулась в норму

### 7️⃣ Экран аккаунта
- [ ] Нажмите на вкладку "Аккаунт" (обычно внизу)
- [ ] Видны ваши данные:
  - Email
  - Статус подписки
  - Кнопка "Управление подпиской"
  - Кнопка "Выход"

### 8️⃣ Темный режим
- [ ] Приложение должно быть в темном режиме (черный фон, неоновые цвета)
- [ ] Текст хорошо читается
- [ ] Цвета соответствуют дизайну (неоновый стиль)

### 9️⃣ Выход из приложения
- [ ] На экране аккаунта нажмите "Выход"
- [ ] Вы вернетесь на экран входа
- [ ] Приложение забыло ваши данные

## ШАГ 6: Проверка на ошибки

### Смотрите логи:
```bash
adb logcat | grep -i "rtrader\|error\|exception"
```

### Ошибки, которые НЕ должны быть:
- ❌ "JSON parse error"
- ❌ "Cannot read property 'accessToken'"
- ❌ "Network error"
- ❌ "Unauthorized"

### Ошибки, которые могут быть (и это нормально):
- ⚠️ "Missing session cookie" - это нормально при первом входе
- ⚠️ "Premature close" - это может быть при переключении сетей
